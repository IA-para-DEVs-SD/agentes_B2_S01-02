# Agente 01 - Scrum Master

## O que faz
Quebra histórias de usuário em tarefas técnicas detalhadas.

## Input
- História de usuário (user story)

## Output
- Lista de tarefas técnicas
- Estimativas de esforço
- Dependências entre tarefas

## Tools utilizadas
- `exa`: Busca informações externas sobre tecnologias
- `break_tasks`: Quebra histórias em tarefas menores
